# Centro Formación Robótica — Web

Landing page para **Centro Formación Robótica (CFR)**, un proyecto en lanzamiento que busca convertirse en el centro de referencia en formación para cirugía robótica en España.

---

## Vista previa

> 🔗 [Ver web en Netlify](https://frabjous-jelly-38ca11.netlify.app)

---

## Sobre el proyecto

CFR es un centro de formación especializado en cirugía robótica, ubicado en Madrid, España. La web actúa como **landing de preventa / lista de espera**, captando el interés de cirujanos antes del lanzamiento oficial en 2026.

---

## Características

- Diseño **responsive** — móvil, tablet y escritorio (4 breakpoints)
- Formulario de **lista de espera** integrado con Formspree
- **SEO optimizado** — Open Graph, Twitter Card, meta keywords
- `robots.txt` y `sitemap.xml` para indexación en Google
- Página de error **404 personalizada**
- Banner de **cookies RGPD** con opciones "Aceptar todo" / "Solo esenciales"
- Botón **WhatsApp** flotante
- Animaciones de entrada con **IntersectionObserver**
